# Student Budget Assistant

A simple Python console app that helps students plan a monthly budget, track expenses, and check whether they can meet a savings goal.

## Features

- Enter monthly income
- Set a savings goal
- Add budget categories and planned amounts
- Record expenses by category
- View a monthly spending report
- See whether your savings goal is achievable
- Get suggestions for where to cut spending

## How It Works

The program:
1. Asks for your monthly income and savings goal
2. Lets you create budget categories such as Rent, Food, and Transport
3. Allows you to enter expenses
4. Compares planned spending with actual spending
5. Shows whether you are under or over budget
6. Suggests cuts if you are short on your savings goal
